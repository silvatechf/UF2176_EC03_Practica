# Endpoint               Método              Ejemplo de URL   
* Alumnos por edad       GET                 (http://localhost:3000/api/consultas/alumnos/por-edad?edad=20)
* Rango de edad          GET                 (http://localhost:3000/api/consultas/alumnos/rango?min=18&max=25)
* Top matriculados       GET                 (http://localhost:3000/api/consultas/cursos/top-matriculados)
* Info matrículas        GET                 (http://localhost:3000/api/consultas/matriculas/info)
* Reporte profesores     GET                 (http://localhost:3000/api/consultas/profesores/detalle)
* Total por curso        GET                 (http://localhost:3000/api/consultas/cursos/total-alumnos)
* Cursos populares       GET                 (http://localhost:3000/api/consultas/cursos/populares?min=1)
